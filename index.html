🔴 Step 11 — Q4 Menu 15 min

⚠️ Different! No button. Two methods outside onCreate but inside the class.

Extra imports: android.view.Menu, android.view.MenuItem

Step 1: Create res/menu/mymenu.xml

<?xml version="1.0" encoding="utf-8"?>
<menu xmlns:android="http://schemas.android.com/apk/res/android">
    <item android:id="@+id/m1" android:title="File" />
    <item android:id="@+id/m2" android:title="Edit" />
    <item android:id="@+id/m3" android:title="Help" />
</menu>

Step 2: Add these two methods in MainActivity.java

public boolean onCreateOptionsMenu(Menu menu) {
    getMenuInflater().inflate(R.menu.mymenu, menu);
    return true;
}
public boolean onOptionsItemSelected(MenuItem item) {
    if (item.getItemId() == R.id.m1)
        Toast.makeText(this, "File", Toast.LENGTH_SHORT).show();
    else if (item.getItemId() == R.id.m2)
        Toast.makeText(this, "Edit", Toast.LENGTH_SHORT).show();
    else
        Toast.makeText(this, "Help", Toast.LENGTH_SHORT).show();
    return true;
}
